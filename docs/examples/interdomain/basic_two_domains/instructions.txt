
Open two sessions of the Mininet server

In the other, run the ovs_configs.sh

From the mininet session, h1 should ping h5


Open three sessions of the sdntrace server

In one load RNP's SDNTrace
cd sdntrace
python run.py --config-file docs/examples/interdomain/sdntrace_rnp.conf

You should see 5 switches

In the second, load CLARA's SDNTrace
cd sdntrace
python run.py --config-file docs/examples/interdomain/sdntrace_clara.conf

You should see 3 switches

Wait for both show the entry "SDNTrace Ready!"

In the third, run your tests
